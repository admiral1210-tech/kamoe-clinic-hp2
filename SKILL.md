# SKILL.md — かもめクリニック HP コンテンツ品質ルール

---

## 【禁止表現】※ AI・ライター共通で冒頭確認必須

- 「必ず治ります」「完治します」など断定的な治癒保証
- 「日本一」「業界最高」など根拠のない最上級表現
- 「今すぐ電話！」「限定」など過度な煽り・緊急誘導
- 「お気軽に」（医療行為に対して軽率な印象を与えるため禁止）
- 「痛くない」「怖くない」など恐怖心を過剰に打ち消す表現
- 体験談・口コミの虚偽引用・誇張
- 患者の特定につながる個人情報の例示

---

## トーン・文体

| 項目 | 方針 |
|------|------|
| 敬体 | 「です・ます」調を統一（体言止め多用禁止） |
| 人称 | 患者向けは「患者さん」「ご家族」。医師・スタッフは「医師」「看護師」 |
| 語調 | 穏やか・誠実・専門的。過度なフレンドリー表現を避ける |
| 漢字 | 医療用語は正確に。読みにくい場合はルビまたは括弧内に読み仮名を添える |
| 数値 | 具体的根拠がある場合のみ記載。推測値・曖昧な数値は禁止 |

## ページ構成ルール

### 診療科ページ（内科・精神科・小児科）

1. **リード文**（100〜150字）：対象患者と診療の特徴を一文で表す
2. **対応症状・疾患リスト**：箇条書き、1項目 30字以内
3. **訪問診療の流れ**：ステップ形式（3〜5ステップ）
4. **よくある質問（FAQ）**：3〜5問。Q は患者視点の自然な疑問文
5. **CTAボタン**：「訪問診療のご相談・お問い合わせ」に統一

### ブログ・お知らせ記事

1. **タイトル**：40字以内。検索意図を含む
2. **リード段落**：記事の要約を 3〜4文で記述
3. **本文**：見出し（H2・H3）で構造化。1見出しあたり 200〜400字
4. **まとめ**：読者の次アクション（問い合わせ・関連ページ誘導）で締める
5. **医療情報の引用**：出典（学会・厚生労働省等）を明記

### エリアページ（area-osaka）

- 各区の地域性に配慮した自然な文章。コピペ感を排除
- 地名・施設名は正式名称を使用
- 「◯◯区 訪問診療」など SEO キーワードを H2 以内に自然に含める

## 採点基準（AI 生成コンテンツのレビュー用）

| 観点 | 配点 | 合格ライン |
|------|:----:|:----------:|
| 禁止表現が含まれていない | 30点 | 30点（必須） |
| トーン・敬体が統一されている | 20点 | 16点以上 |
| ページ構成ルールに沿っている | 20点 | 14点以上 |
| 医療情報の正確性・出典あり | 20点 | 14点以上 |
| SEO キーワードが自然に含まれる | 10点 | 6点以上 |
| **合計** | **100点** | **80点以上で公開可** |

> 禁止表現が1つでも含まれる場合は、合計点によらず公開不可とする。
