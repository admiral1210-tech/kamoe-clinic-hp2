# 画像ファイル配置ガイド

このディレクトリに以下のファイルを配置してください。

## 必須ファイル

| ファイル名                   | 用途                   | 推奨サイズ                   |
| ---------------------------- | ---------------------- | ---------------------------- |
| `logo.png` または `logo.svg` | ヘッダー・フッターロゴ | 横幅 200px 以上、透過PNG推奨 |
| `favicon.ico`                | ブラウザタブアイコン   | 32×32px                      |
| `og-image.jpg`               | SNSシェア時のOGP画像   | 1200×630px                   |

## 任意ファイル（実際の写真に差し替え）

現在は Unsplash の仮画像を使用しています。
以下のファイルを配置すると index.astro の各セクションに反映されます。

| ファイル名         | 使用箇所                    |
| ------------------ | --------------------------- |
| `hero.jpg`         | ヒーローセクション背景      |
| `about.jpg`        | 訪問診療とは セクション     |
| `dept-naika.jpg`   | 内科 セクション             |
| `dept-seishin.jpg` | 精神科 セクション           |
| `dept-shoni.jpg`   | 小児科 セクション           |
| `medical-pro.jpg`  | 医療・介護関係者 セクション |
| `recruit-hero.jpg` | 採用ページ ヒーロー         |

配置後は `index.astro` / `recruit.astro` の各 `image.src` を
`~/assets/images/ファイル名` に変更してください。
